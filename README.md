# HH Auto: Тестовая среда

## Workflow Откатов

🔴 Проблема в коде → `iterations.md`
↓
🔍 Выбор последней стабильной версии
↓
📂 Восстановление из `/backups/YYYY-MM-DD/`
↓
🔄 Возврат к работоспособному состоянию

### Ключевые точки

- Каждое значимое изменение → бэкап
- `iterations.md` — история изменений
- Бэкапы в `/backups/` по датам
- Минимум правил, максимум прозрачности

### Приоритет

1. Быстрый откат
2. Минимальные потери
3. Сохранение контекста разработки
